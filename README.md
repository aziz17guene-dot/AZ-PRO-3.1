## 📱 Fonctionnalités

### 1. **Point de Vente (POS)**
- Vente rapide par code-barres
- Calcul automatique des bénéfices
- Historique des ventes détaillé
- Factures/reçus thermiques

### 2. **Gestion de Stock**
- Inventaire en temps réel
- Alertes rupture de stock
- Mouvement de stock (entrée/sortie)
- Archivage des ventes anciennes

### 3. **Gestion Financière**
- Bénéfices/pertes par produit
- Dépenses d'exploitation
- Résumés journaliers/mensuels
- Graphiques de tendance

### 4. **Multi-Boutique**
- Gestion de plusieurs magasins
- Utilisateurs avec rôles
- Synchronisation entre caisses
- Rapports consolidés

### 5. **Fonctionnalités Avancées**
- Application Web (PWA)
- Fonctionne hors-ligne
- Installation sur téléphone/PC
- Sauvegarde/restauration JSON
- Audit trail complet

## 🚀 Installation & Utilisation

### Installation Rapide
```bash
1. Cloner ou télécharger:
   https://github.com/aziz17guene/AZIZ_PRO-V2.0

2. Ouvrir index.html dans navigateur

3. Ou utiliser lien direct:
   https://aziz17guene-dot.github.io/AZIZ_PRO-V2.0/
```

### Sur Android
```
1. Ouvre le lien dans Chrome
2. Menu → "Ajouter à l'écran d'accueil"
3. App s'installe comme native
```

### Sur PC/Mac
```
1. Ouvre le lien dans navigateur
2. Peut être installée comme app
3. Fonctionne hors-ligne
```

### Connexion Initiale
```
Email: n'importe quel email
Code: 1010 (ou autre numérique)
```

## 📋 Configuration Système

### Fichiers Requis

- JavaScript activé

- Connexion Internet (pour première visite)
